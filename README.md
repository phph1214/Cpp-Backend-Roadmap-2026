# C++ Backend Developer Roadmap (2026)

Targeting Top-tier Tech Companies by July 2026.
Environment: Ubuntu 24.04 (WSL) | GCC | CMake

## Learning Modules
1. **Modern C++**: Focusing on C++11/14/17 standards (RAII, Move Semantics).
2. **STL**: Understanding internal implementation (RB-Tree, Hash Table).
3. **System**: Linux Memory Model, Multi-threading, Network I/O (Epoll).
4. **Projects**: High-performance server implementation.
